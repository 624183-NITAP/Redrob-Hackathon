## Overview

This repository contains our solution for the Redrob Hackathon candidate ranking challenge.

The objective is to identify and rank the top 100 candidates from the provided candidate dataset according to their relevance to the released job description. The solution processes candidate profiles, extracts meaningful signals, computes ranking scores, and generates a submission-ready CSV file.

## Methodology

The ranking pipeline consists of the following stages:

### 1. Data Loading

Candidate profiles are loaded from the provided `candidates.jsonl` dataset.

### 2. Feature Extraction

Relevant profile information is extracted, including candidate experience, skills, qualifications, and other ranking signals.

### 3. Candidate Scoring

Each candidate is assigned a score based on extracted features and alignment with job requirements.

### 4. Ranking

Candidates are sorted according to their computed scores.

### 5. Submission Generation

The top 100 candidates are selected and written to the final submission CSV.

## Dataset

The original dataset file is not included in this repository because of its size.

Place the provided dataset at:

```text
data/candidates.jsonl
```

---

## Reproducing the Submission

After placing the dataset in the `data` directory:

```bash
python src/rank.py
```

This command generates the final ranked submission file.

## Output

The final submission file is:
